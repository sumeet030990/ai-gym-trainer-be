## Run the Program Using
- uv run uvicorn main:app --reload

## UV commands
- uv init: Create a new Python project.
- uv add: Add a dependency to the project.
  - Ingest your existing dependencies straight into your pyproject.toml configuration:
    - uv add -r requirements.txt
- uv remove: Remove a dependency from the project.
- uv sync: Sync the project's dependencies with the environment.
- uv lock: Create a lockfile for the project's dependencies.
- uv run: Run a command in the project environment.

## Alembic commands
- Create an autogenerated revision or say create migration files
  - alembic revision --autogenerate -m "Name of the migration eg: create_inital_database"
- Apply the newest revision
  - alembic upgrade head
- Roll back the last revision
  - alembic downgrade -1
